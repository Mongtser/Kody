# Smart Gun Case Arduino Prototype

This folder contains a public-safe version of the Arduino sketch used for the Smart Gun Case embedded systems project.

The code demonstrates the prototype architecture:

- fingerprint sensor setup and matching flow
- keypad input handling
- LCD user feedback
- servo-based lock actuation
- menu flow for enrollment, database deletion, and locking

## Public-Safe Notes

The original project was a course prototype, not a production security product. For public portfolio use, access codes are redacted and should be replaced with new values before any private testing.

Do not reuse the placeholder passwords in this repository. The sketch is included to show system integration and control-flow logic, not to provide a secure access-control implementation.
